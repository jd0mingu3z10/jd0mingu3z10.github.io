Why the “s” in HTTPs is important

In this article from ComputerWorld.com, it explains the importance in using a secure HTTP (Hypertext Transfer Protocol) when using wireless in a public place.

The S in HTTPS stands for secure, when you connect to a server using HTTPS, your HTTP requests using secure sockets layer (SSL). The SSL is a protocol used to secure communications between a client and server. The protocol creates an encryption to keep a hacker from hijacking your information.

Websites that offer a secure connection, a few steps need to happen in the process before securing that connection. First the browser resolves the website's name to an IP address using the domain naming service (DNS), then have communication with the DNS server you will many times be redirected to a secure connection. Even if you typed http://. Server replies to the HTTP request with a server certificate. The certificate contains information to setting up the encryption. The type of server certificate is called certificate authority, and the issuing authority is digicert. Digicert is trusted by major browsers such as Explorer, Chrome and Firefox. A trusted root certificate is already in your browser that certifies and verifies that the certificate presented for the secured website can also be trusted. Once it's all verified, it creates an SSL tunnel to your connection, given the client and server an encrypted tunnel so that you are now securely connected.

One of the examples was explaining how unsecured it can be to use HTTP in a hotspot which normally is offered at a coffee shop. Reason that the hotspot needs to be unsecured is so that a guest network can be shared for its customers. For example if you are in that coffee shop connected using HTTP and want to look into your online banking. If you are not sure who you're connecting to and it does not have an SSL connection, information may be compromised if someone were to be nearby with a laptop running Unix. So as you're entering your credentials, that person may be able to collect that information and you can be compromised.
